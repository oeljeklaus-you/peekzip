In-App Purchases

PeekZip Pro is a non-consumable in-app purchase. If you have already purchased PeekZip Pro, use the Restore Purchase button inside the app to restore access.

File Access

PeekZip only accesses archive files and output folders that you select. If a file cannot be opened or extracted, please make sure the file is not damaged and that you have permission to access it.
